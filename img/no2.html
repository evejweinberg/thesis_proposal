Johnny Appleseed Branding
Working with Mekanism, Eve had the exciting privilege of defining Johnny Appleseed's visual identity and online presence. Working off of the brand line 'Refreshingly Sweet and Intense,' (and some existing package artwork) we got to work to create a website, web banners, intro animation, and social media imagery. The first step was to create a teaser website, with an enticing interactive animation.
